# ⚙️ DevOps Flow | CI/CD Pipeline Visualizer

An engineering-focused visualization tool for software lifecycle management and Continuous Integration/Continuous Deployment (CI/CD) pipelines.

## ✨ Features
- **Node-Based Pipeline**: Visual representation of "Build", "Test", "Deploy", and "Monitor" stages.
- **Status Indicators**: Real-time status icons (Checkmarks, Spinners, Pending nodes).
- **Linear Progress UI**: Connected pipeline flows with smooth CSS transitions.
- **Developer-Centric Theme**: Dark mode "JetBrains Mono" aesthetic for technical clarity.

## 🛠️ Tech Stack
- **Languages**: HTML, CSS.
- **Concepts**: Software Engineering, Automation, Lifecycle Management.

## 🧠 Why I built this?
Modern development is impossible without DevOps. I built this to visualize the automated flow of code from a local machine to a production server, highlighting my understanding of the software engineering pipeline.
